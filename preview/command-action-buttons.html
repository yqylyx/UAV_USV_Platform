指令队列 — 操作按钮（均匀填满）

按钮用 flex: 1 1 0 等分 240px 操作列，按钮数量不同时自动均匀分布。
高度 26px · 字号 12px · 图标 13px · gap 4px，与标准表格行高协调，不多不少刚好填满。

方案一（均匀填满版） 弹性等分 · 任何数量自动填满
状态 目标 / 区域 操作（240px）
RUNNING 灯塔 A / 北部
PAUSED 信标 B / 东部
READY 浮标 3 / 外海
DRAFT 浮标 X / 锚地
COMPLETED 灯塔 A / 北部
设计要点：
1. .action-row { display: flex; gap: 4px; width: 100%; } — 弹性容器，强制单行
2. .btn { flex: 1 1 0; justify-content: center; } — 按钮等分剩余宽度，内容居中
3. 高度 26px（在 8px padding 的表格行中上下各有 3px 呼吸空间）
4. 字号 12px、图标 13px —— 与 Element Plus 默认表格字号一致，不大不小
5. 3 个按钮时各占约 70px，6 个按钮时各占约 33px，均匀填满 240px
6. 动态操作按钮（暂停/完成/异常/恢复/启动/取消/准备）不带图标，仅文字
7. hover 翻转色延续暗色主题，按语义着色（primary/success/warning/danger）
