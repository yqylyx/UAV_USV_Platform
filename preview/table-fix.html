表格选中 / 悬停 背景色 — 仅悬停变蓝灰

选中行（current-row）不做背景变化，保持现有透明深色背景，与普通行一致。
只有鼠标悬停时显示蓝灰半透明反馈 rgba(160, 180, 200, 0.08)。

添加到 main.css 的代码

/* 表格 hover — 蓝灰色，仅悬停反馈 */
.console-table .el-table__body tr:hover > td {
  background-color: rgba(160, 180, 200, 0.08) !important;
}

/* 选中行不额外变色，保持透明 */
.console-table .el-table__body tr.el-table__current-row > td {
  background-color: transparent !important;
}
💡 选中行保持透明背景，只有鼠标划过时才显示蓝灰反馈。
这样既保留了当前行的逻辑选中状态（Element Plus 内部仍可识别），又不会破坏视觉统一。
