# UAV-USV Platform Frontend

Vue 3 + TypeScript + Vite 前端工程，用于 UAV-USV 海空协同仿真与任务控制平台。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- Element Plus
- Lucide Icons

## 当前页面

- 登录页
- 系统总览
- 设备管理
- 运行监控

## 页面设计原则

- 页面优先围绕任务闭环设计，不做普通后台堆表格。
- UAV、USV、ROS、Unity 等对象应以任务资产和态势节点方式展示。
- 查询、筛选、按钮、状态等控件需要保持紧凑和可扫描。
- 与 PPT 对齐，优先形成深蓝科技风格的任务控制台。

- 通信网络监控
- 协同围捕过程展示
- 实验评估与回放
